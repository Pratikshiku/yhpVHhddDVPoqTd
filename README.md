## Campus Job Recruitment System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/3a02db1e6cdb4cd7837819dd56d024f9/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): job seekers, enterprises, administrators.

**Job Seeker Functions:**
1. Login
2. Register
3. Homepage
4. Forum
5. Enterprises
6. Announcement Information
7. Job Recruitment
8. Personal Center (change password, modify personal information)
9. Job Collection
10. Interview Application
11. Messages
12. Secure Exit

**Enterprise Functions:**
1. Login
2. Register
3. Homepage
4. Personal Center (change password, modify personal information)
5. Resume Management
6. Job Recruitment Management
7. Interview Application Management
8. Forum Management
9. Announcement Information Management
10. Secure Exit

**Administrator Functions:**
1. Login
2. Homepage
3. Personal Center (change password, modify personal information)
4. Administrator Management
5. Enterprise Management
6. User Management
7. Resume Management
8. Job Recruitment Management
9. Interview Application Management
10. Basic Data Management
11. Announcement Information Management
12. Forum Management
13. Secure Exit
